* Use the prompts and talking points (ðŸ”‘) below to review the following key points:

  * âœ”ï¸ Adjust start and end positions

  * âœ”ï¸ The `while` loop

  * âœ”ï¸ Exit conditions

* Open `04-Binary-Search/Solved/index.js` in your IDE and explain the following:

  * ðŸ”‘ We start by getting the start and end positions, which we will update when we determine what half our value lies in:

    ```js
    let start = 0;
    let end = array.length - 1;
    ```

  * ðŸ”‘ We use a `while` loop to continuously run the functionality of getting a middle value, checking if the searched value is greater than, less than, or equal to it, and adjusting the start and end points if we need to:

    ```js
    while (start <= end) {
      let mid = Math.floor((start + end) / 2);

      if (array[mid] === element) {
        return true;
      } else if (array[mid] < element) {
        start = mid + 1;
      } else {
        end = mid - 1;
      }
    }
    ```

  * ðŸ”‘ We are done running this function when either the middle value is the value we're searching for or we've cut the search array down so much that the end value becomes less than the start value and the `while` loop ends to return `false`.

# Module 01 Required Session: HTML-Git-CSS

## Overview

In today's required session, students will review CSS, relative paths, and deploying to GitHub Personal Pages.

## Learning Objectives

By the end of class, students will be able to:

- Use CSS to style and layout a simple HTML 'portfolio'

- Implement relative paths to connect files between directories

- Deploy a project using GitHub Personal Pages.

## Instructor Notes

- In today's class, you will introduce and demo the module challenge. Find time prior to class to install any dependencies, read the acceptance criteria, and familiarize yourself with the application functionality so you are prepared to answer any questions.

* The solution to the first activity requires floats. Students wlll likely struggle finding it. If it's been awhile, you may want to refresh your knowledge on floats in order to improvise demos and field any questions.

* If you are unfamiliar with GitHub Pages, you will want to perform a dry run of a deployment prior to class. Consult the [GitHub Pages](https://pages.github.com/) documentation for instructions.


* Ask the class the following questions (â˜) and call on students for answers (ğŸ™‹):

    * â˜ï¸ What is the first step to solve this problem?

    * ğŸ™‹ In order to move things in the array, it might be easiest to create a new clone of the array that we can modify! We can do that with `.slice(0)`. 

    * â˜ Great idea! So something like `let temp = arr.slice(0)` would get that situated for us. What's next?

    * ğŸ™‹ We'll need a loop that goes through the input array as many times as the value of `n` and grabs the first item in the array and moves it to the end. 

    * â˜ Exactly. Let's take a look at how to do that. In this code, we remove the first item of the array, and store it in a temporary variable `first`, and then use the `.push()` method to take that variable and _push_ it on to the end of the array.

    ```js

        let first = temp.shift();
        temp.push(first);
    }
    ```

    * â˜ And don't forget we need to return the value at the end: `return temp`. And with that, we're all done!

* Full solution:

```js
function arrayRotation(arr, n) {
    let temp = arr.slice(0);

    for (let i = 0; i < n; i++) {
        let first = temp.shift();

        temp.push(first);
    }

    return temp;
}
```

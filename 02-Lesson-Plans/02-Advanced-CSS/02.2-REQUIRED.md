# Module 02 Virtual Class (Required): Advanced CSS

## Overview

In today's required virtual class, students will review CSS Grid, pseudo-classes, and CSS variables.

## Learning Objectives

By the end of class, students will be able to:

* Implement a layout using CSS Grid

* Use pseudo-classes to style very specific elements

* Use CSS variables for easier style changes within their CSS

### 5. Instructor Review: CSS Grid (10 min)

* Bring students back from breakout rooms.

* Use the prompts and talking points below to review the following key (ğŸ”‘) points:

  * âœ”ï¸ Use grid value in display property

  * âœ”ï¸ Use `grid-template-columns` and `rows` for layout

  * âœ”ï¸ Specifying sizes of elements

  * âœ”ï¸ Selectors to target specific elements

* Open [3-Stu_CSS-Grid/Solved/assets/css/style.css](../../01-Class-Content/02-Advanced-CSS/01-Activities/3-Stu_CSS-Grid/Solved/assets/css/style.css) in your IDE and explain the following:

  * ğŸ”‘ First we target the div with the class of `.grid-container` and give it `display: grid`, and specify `grid-template-columns` and `grid-template-rows`. We will use the second 2 rules to dictate the size of our grid in rows/columns

  ```css
  .grid-container {
    display: grid;
    grid-template-columns: repeat(8, 1fr);
    grid-template-rows: repeat(8, 1fr);
  }
  ```

  * Now that we have our ten items across two rows. Lets space them out evenly and make it look a little nicer. We can do that by adding a `grid-gap` rule.

  ```css
  .grid-container {
    display: grid;
    grid-template-columns: repeat(8, 1fr);
    grid-template-rows: repeat(8, 1fr);
    grid-gap: 20px;
  }
  ```

  * ğŸ”‘ Now that we have our basic layout, lets target elements and make them occupy the spaces specified in the instructions. Starting with grid-item 10, lets make it two rows spanning three columns using `grid-column` and `grid-row` rules.

  ```css
  .item-10 {
    grid-column: 2 / span 3;
    grid-row: 2 / span 4;
  }
  ```

  * ğŸ”‘ We write CSS to specify that every third item (3, 6, and 9) should span three columns and span two rows using the same rules we did for the grid-item 10. To specify every 3rd element, we use `grid-item:nth-child(3n)` selector and adding the appropriate rules.

  ```css
  .grid-item:nth-child(3n) {
    grid-column: span 3;
    grid-row: span 2;
  }
  ```

* Ask the class the following questions (â˜ï¸) and call on students for the answers (ğŸ™‹):

  * â˜ï¸ When using `grid-column` or `grid-row`, what does `X / span Y` mean?

  * ğŸ™‹ These two rule names are shorthand for specifying the `start` and `end` positions. So when providing two integers, we are specifying the `start` and `end` of the column or row.

  * â˜ï¸ What can you do if you don't completely understand this?
